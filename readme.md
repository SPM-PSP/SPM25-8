# 个人健康管理系统（Personal Health Management System, PHMS）

一个面向用户健康数据管理与健康内容互动的全栈式健康管理平台。系统支持健康数据记录、智能可视化、健康资讯获取与个性化推荐，旨在通过数据+社交+提醒的闭环提升用户健康意识和自我管理能力。

---

## 📚 项目背景

近年来，随着慢性疾病年轻化趋势和全民健康意识的提升，传统健康类应用普遍存在以下痛点：

- **重记录，轻分析**：只支持健康指标记录，缺乏个性化反馈。
- **功能单一，缺乏互动**：用户缺乏知识获取和社区支持渠道。
- **缺乏预警机制**：难以及时干预潜在健康风险。

本项目通过构建健康指标记录、动态健康模型、互动社区与智能提醒功能模块，打造一个全场景、智能化、社交化的健康管理平台。

---

## 🧱 技术架构

- **后端**：SpringBoot + SpringMVC + MyBatis
- **前端**：Vue.js + Vue Router + Axios
- **数据库**：MySQL 8.x
- **安全机制**：基于JWT的用户身份认证

---

## 👥 用户角色

### 🔐 管理员用户
- 用户管理（增删查改，启用/禁用）
- 标签分类管理
- 健康资讯管理（发布、审核、推荐）
- 健康模型配置管理（如血压、血糖等）
- 健康记录全览与统计
- 评论与消息内容管理
- 数据可视化仪表盘

### 👤 普通用户
- 登录 / 注册 / 修改密码
- 浏览健康资讯与收藏、搜索
- 根据模型记录自身健康指标（如体重、步数等）
- 查看健康可视化分析（趋势图、评分等）
- 接收指标预警、评论提醒等消息
- 编辑个人信息

---

## 📌 核心功能一览

| 模块                 | 说明                                                  |
|----------------------|-------------------------------------------------------|
| 用户认证             | 支持注册、登录、修改密码，JWT身份令牌认证           |
| 健康资讯             | 浏览/评论/点赞/收藏资讯，支持标签分类与搜索         |
| 健康模型             | 管理个人健康指标（如体重、血压、心率等）           |
| 可视化分析           | 生成图表展示个人指标变化趋势、评分、风险预警等     |
| 消息中心             | 异常提醒、评论互动、系统公告等推送                  |
| 后台管理             | 管理员可操作所有用户数据、评论内容、资讯内容等     |
